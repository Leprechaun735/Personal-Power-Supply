# Personal-Power-Supply
I grow tired of not having the right power I need for running things so I am taking matters into my own hands

Currently a work in progress
I am using an old 65W computer charger wired into an adjustable LM2596.
I desoldered the potentiometer that it came with to set is up with a new bulkier one that I can change with my hand and not a screwdriver. 

Still to be completed
  Encasement: Half way through turning an old beer case into a SAFE enviornment for the circuit to stay in
  I need my multimeter to read the voltage but I want to get a Voltmeter Ammeter LED Dual Volt Amp Meter Voltage Current Gauge Q3E8 soon so I can wire that into the system
